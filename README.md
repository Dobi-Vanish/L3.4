# Выполненное задание L3.4

Для запуска:  
1. Клонируйте репозиторий: `git clone https://github.com/Dobi-Vanish/L3.4`  
2. Перейдите в скопированный репозиторий  
3. Запустите через `make all`

Для просмотра перейдите на сайт http://localhost:8080/  
Процесс обработки изображения может занять время.  
Для просмотра локального хранения внутри контейнера выполните команды:  
`docker exec -it deployments-api-1 sh`  
`ls -la /app/storage`  

Водяной знак отображается в правом нижнем углу в виде надписи "SAMPLE"
